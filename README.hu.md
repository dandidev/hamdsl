# HamDsl

Típusbiztos Kotlin könyvtár rádióamatőr számításokhoz.

A HamDsl egy tanulásközpontú projekt, amely a rádióamatőr és elektronikai számítások gyakorlati alkalmazását és megértését támogatja.

A könyvtár fejlődése során folyamatosan új képletekkel, mértékegységekkel és számítási segédletekkel bővül.

## Funkciók

* SI mértékegységek
* dB és dBm számítások
* Teljesítmény, feszültség, áram és ellenállás számítások
* Szabadtéri terjedési csillapítás (FSPL)
* Hullámhossz számítás
* Típusbiztos operátorok
* Példaszámítások

## Projekt állapota

A HamDsl aktív fejlesztés alatt áll.

A könyvtár jelenleg a rádióamatőr és elektronikai számításokhoz szükséges alapvető fizikai mennyiségekre és összefüggésekre fókuszál. A jövőben további számítások és segéd függvények kerülnek beépítésre.

A projekt fejlődése szorosan kapcsolódik a rádióamatőr tanulmányokhoz, ezért az új funkciók fokozatosan jelennek meg.

## Példák futtatása

A projekt példái az alábbi formákban érhetők el:

* Kotlin script
* Kotlin Notebook

A példák futtathatók IntelliJ IDEA környezetből vagy bármely Kotlin Notebook kompatibilis környezetből.
